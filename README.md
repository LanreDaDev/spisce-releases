# Spisce

> AI Video Series Production Desktop App

## ✨ What is Spisce?

Spisce helps you organize AI video production with:

- **12 Specialized Tools** - From narrative planning to final assembly
- **Project Hierarchy** - Projects → Series → Episodes
- **Artifact Tracking** - Track dependencies between tools and outputs
- **Batch Operations** - Generate 30 shots in 15 minutes instead of 3 hours
- **Cost Tracking** - Monitor API costs for each operation
- **Local-First** - Your data stays on your machine

### Built For
- Storytellers using API-based AI tools (OpenAI, Anthropic, ElevenLabs, Fal.ai)
- Creators who want organization without complexity
- Anyone making AI-powered video series

## 📦 Installation

### macOS
1. Download the `.dmg` file
2. Open it and drag Spisce to Applications
3. Right-click Spisce and select "Open" (first time only)
   - You'll see a security warning (we're not code-signed yet)
   - Click "Open" to confirm

### Windows
1. Download the `.exe` file
2. Run the installer
3. You may see a SmartScreen warning (we're not code-signed yet)
   - Click "More info" → "Run anyway"

### Linux
1. Download the `.AppImage` file
2. Make it executable: `chmod +x Spisce.AppImage`
3. Run it: `./Spisce.AppImage`
